# Data Preprocessing Pipeline

## Project Overview
This project demonstrates a complete data preprocessing pipeline using Python.

## Features
- Data Cleaning
- Missing Value Imputation
- Mean Imputation
- Median Imputation
- KNN Imputation
- Outlier Detection using Z-Score
- Outlier Detection using IQR
- Memory Optimization
- Data Type Conversion
- One-Hot Encoding
- Target Encoding

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- SciPy
- Category Encoders

## Dataset
A sample dataset containing Age, Salary, Gender, City, and Target columns.

## How to Run

1. Install dependencies:
```
pip install -r requirements.txt
```

2. Run the project:
```
python preprocessing_pipeline.py
```
